2. Average waiting time:
	
	* FCFS:
		wP1 = 0
		wP2 = 8-0.4 = 7.6
		wP3 = 12-1 = 11
		wP4 = 13-1.3 = 11.7			-> Average waiting time = (0 + 7.6 + 11 + 11.7) / 4 = 7.575

	* SJF:
		wP1 = 0
		wP2 = 8-1 = 7
		wP3 = 9-0.4 = 8.6
		wP4 = 13-1.3 = 11.7			-> Average waiting time = ( 0 + 7 + 8.6 + 11.7 ) / 4 = 6.825

	* SRTF:
		wP1 = 0
		wP2 = 0.4-0.4 = 0
		wP3 = 0.4+0.6-1 = 0
		wP4 = (0.4+0.6+1+3.4)-1.3 = 4.1		-> Average waiting time = ( 0 + 0 + 0 + 4.1 ) / 4 = 1.025

	* RR:
		wP1 = 0
		wP2 = 1-1.4 = 0.6
		wP3 = 2-1 = 1
		wP4 = 3-1.3 = 1.7			-> Average waiting time = ( 0 + 0.6 + 1 + 1.7 ) / 4 = 0.825
